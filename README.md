# TTS2CUBE-Pico

TTS2CUBE-Pico addon.

## Development

1. Init dev environment

    ```
    ./scripts/bootstrap.sh
    ```

2. Start server

    ```
    ./scripts/start-server.sh
    ```

3. Start web dev server

    ```
    ./scripts/start-web.sh
    ```

## Build

Run the following command to build this addon.

```
./scripts/build.sh
```
